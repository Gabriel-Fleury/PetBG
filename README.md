  PetBG 🐾 - Barra do Garças
Um projeto de impacto social desenvolvido para ajudar a comunidade de Barra do Garças (MT) a encontrar animais perdidos,
conectar quem encontrou um pet ao seu dono e facilitar o processo de adoção responsável.


                       Status do Projeto: 🚀 Em desenvolvimento (Projeto Pessoal)

                                              
  📋 Sobre o Projeto
O PetBG nasceu da vontade de usar a tecnologia para resolver um problema real da minha cidade.
A plataforma permite que moradores anunciem pets perdidos, achados ou disponíveis para doação de forma rápida e organizada.

  Principais Funcionalidades:
-Anúncios Categorizados: Separação visual por cores para Urgência (Perdidos), Atenção (Achados) e Esperança (Adoção).
-Autenticação: Sistema de login seguro para gestão dos anúncios.
-Gestão de Imagens: Upload obrigatório de fotos para facilitar a identificação.
-Limpeza Automática: Os anúncios expiram após 30 dias, mantendo a plataforma sempre atualizada.

  🎨 Identidade Visual & UX
O projeto utiliza um sistema de cores intuitivo para facilitar a navegação:
Categoria     Cor    Significado
